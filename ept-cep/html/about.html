Project Overview

This is a simple web application for calculating design parameters for standard transmission line geometries as a complex engineering project for the Electrical Power Transmission (EE-362) course in partial fulfillment of the requirements for BE-Electrical Engineering, NEDUET.

The application first asks the user to select one of ten practical, realistic transmission line geometries. The user will then specify values for the geometry's various input parameters, such as:

  • Transmission voltage
  • Transmission distance
  • ACSR Conductor Code
  • Transmission Line Length
  • Number of conductors per bundle

The program will then calculate the following output parameters

  • Resistance per unit length
  • Capacitance per unit length
  • Inductance per unit length
  • Resistive losses
  • Line current
  • Corona losses
  • SIL
